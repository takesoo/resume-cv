### 株式会社カンブライト (2021/4〜現在)

#### 製造管理システム受託開発（2025/7〜現在）

**概要**: 顧客の製造管理業務をデジタル化するDXプロジェクト

**課題**:
- 紙ベースの業務で膨大な作業時間とヒューマンエラーの頻発、それに伴う経済的損失
- 既存システムでは顧客独自の業務フローに対応できない

**取り組み**:
- 顧客業務のヒアリング・分析から要件定義までを主導
- Claude CodeとCursorを活用し、1人で要件定義〜プロトタイプ開発〜本開発を実施
- tRPCによる型安全なAPI設計で開発効率とコード品質を両立
- Spec Kitを活用したSDD

**成果**:
- AI活用により、プロジェクトリードとフルスタックエンジニアを1人で兼任
- 顧客との認識合わせからプロトタイプ構築までを迅速に実行し、フィードバックサイクルを短縮

**技術**: Next.js(App Router), TypeScript, tRPC, Supabase, Tailwind CSS, Shadcn/UI, Claude Code, Cursor, Spec Kit

#### 食品製造業向けクラウドERP開発（2021/4〜2025）

**概要**: HACCP準拠の衛生記録・製造記録管理、生産管理、在庫管理、受注管理、請求書発行など、食品製造卸業に必要な機能を搭載したクラウドERPの開発

**課題**:
- ユーザーの課題
  - 業界特有の規制（HACCP）に対応したシステムが少ない
  - 各社で異なる業務フローへの柔軟な対応が必要
- 社内の課題
  - プロダクトの成長と現在地を大雑把な定性分析でしか把握していない

**取り組み**:
- フロントエンド: Reactコンポーネント設計、UI実装
- バックエンド: DB設計、API設計・実装、テストカバレッジ改善
- データ分析: 本番DBのデータ分析、BIダッシュボード構築による、プロダクトの状況を可視化

**成果**:
- **20社以上の食品製造業者に導入**
- 顧客の業務効率化とHACCP対応の両立を実現
- BIダッシュボードによりプロダクトの状況把握のためのデータ可視化を提供
- 過去の分析結果と比較したデータ分析レポートを、月に一回slackで発信し、定量的な成果を意識する文化を組織に定着

**技術**: Ruby on Rails, Next.js(Pages Router), TypeScript, AWS Aurora, AWS ECS, OpenAPI, DuckDB, Evidence
